📱 OVERVIEW
-----------
This is a React Native Expo application that combines mood analysis, music recommendations, 
and cooking suggestions. The app helps users discover food and music based on their current 
emotional state using AI-powered mood detection.

🔧 SYSTEM REQUIREMENTS
----------------------
Before starting, ensure you have the following installed:

1. Node.js (version 18.0.0 or higher)
   - Download from: https://nodejs.org/
   - Verify installation: node --version

2. npm (comes with Node.js) or yarn
   - Verify npm: npm --version
   - Or install yarn: npm install -g yarn

3. Git (for version control)
   - Download from: https://git-scm.com/
   - Verify installation: git --version

4. Expo CLI (recommended)
   - Install globally: npm install -g @expo/cli
   - Verify installation: expo --version

📦 INSTALLATION STEPS
---------------------

Step 1: Clone or Download the Project
-------------------------------------
If using Git:
git clone [repository-url]
cd "mood music, cook/project"

Or download and extract the project folder.

Step 2: Install Dependencies
----------------------------
Navigate to the project directory and run:

npm install

Or if using yarn:
yarn install

This will install all required dependencies including:
- React Native and Expo framework
- Navigation libraries
- UI components (Lucide icons, Linear Gradient)
- Animation libraries (Reanimated)
- Font libraries (Inter, Poppins)
- And many more...

Step 3: Start the Development Server
------------------------------------
Run one of the following commands:

For normal start:
npm start
or
expo start

For clean start (clears cache):
npm run dev
or
expo start --clear

For complete reset:
npm run reset
or
expo start --clear --reset-cache

🌐 ACCESSING THE APPLICATION
----------------------------

Web Browser:
- After starting the dev server, press 'w' in the terminal
- Or open http://localhost:8081 in your browser
- The app runs perfectly in web browsers

Mobile Device (Expo Go):
1. Install Expo Go app from App Store/Google Play
2. Scan the QR code shown in terminal
3. The app will load on your device

iOS Simulator (macOS):
- Press 'i' in the terminal to open iOS simulator

Android Emulator:
- Press 'a' in the terminal to open Android emulator

⚠️ TROUBLESHOOTING
------------------

Common Issues & Solutions:

1. "Metro bundler error" or "Cache issues":
   Solution: Run npm run reset or expo start --clear --reset-cache

2. "Module not found" errors:
   Solution: Delete node_modules and run npm install again

3. "Expo CLI not found":
   Solution: Install globally with npm install -g @expo/cli

4. Images not loading:
   Solution: Ensure all images are in the correct img/ directory

5. "Port already in use":
   Solution: Kill the process or use a different port with expo start --port 8082

6. Font loading issues:
   Solution: Restart the development server
